# Elektro W. Klaffel e.U. – Website (Testseite)

Statische Website (HTML/CSS/JS, keine Server-Verarbeitung) für **Elektro W. Klaffel e.U.**

**Live (GitHub Pages):** https://dherzog90.github.io/klaffel.at/

## Aufbau
- `index.html` – Einseitige Website (Hero, Leistungen, Kontakt, Impressum)
- `styles.css` – Markenanpassungen auf Basis von Pico.css
- `assets/css/pico.min.css` – CSS-Framework (selbst gehostet, MIT-Lizenz)
- `assets/fonts/` – Selbst gehostete Schriften: Cabin (OFL), Inconsolata (OFL)
- `assets/js/contact.js` – Kontaktformular öffnet das Standard-E-Mail-Programm (`mailto:`)
- `logo.png`, `elektrotechnik.png` – Firmenlogo und Gütesiegel

## Hinweise
- Alle Assets werden lokal ausgeliefert (kein CDN, DSGVO-freundlich).
- Schriften: Originale (Gill Sans MT / Consolas) für Windows-Nutzer, sonst Cabin / Inconsolata als Open-Source-Fallback.
- **Offen:** UID-/Firmenbuchnummer im Impressum ergänzen.
